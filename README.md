
## Instructions
- clone git repository.
- cd into project folder.
- run `composer install`.
- run `npm install`.
- run `run dev`.
- run `php artisan migrate:fresh --seed`.
- in different terminal run `php artisan serve`
- open `localhost:8000` in browser
